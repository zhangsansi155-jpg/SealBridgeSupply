## 使用原则

- 产品页承接采购和询价意图，技术文章承接问题型、标准型和材料型长尾词。
- 不因为单日排名波动改标题或重复发布相似页面；连续 7–14 天趋势才作为主要调整依据。
- “站内已覆盖”不等于“Google 已收录或已有排名”，每日检测必须分别记录页面状态、收录状态、展示和排名。
- 不宣称单个垫片具备 IP、NEMA 或整机认证；相关等级属于完整装配和测试条件。
- 没有真实产品、材料、工艺或质量体系支撑的关键词不建立落地页。

## 暂不建立 SEO 页面

- EMC、EMI、RFI：没有真实导电屏蔽密封产品前不做。
- FIPG、CIPG、PIPG：没有确认实际点胶能力前不做。
- PPAP、APQP、IMDS：没有真实汽车质量流程前不做。
- FKM、FVMQ、TPV、TPE：没有稳定供应及真实数据表前不做。
- CE gasket：普通垫片不应作为独立 CE 产品宣传。

## 每日检测流程

1. 检查首页、P0 目标页和新发布文章是否返回 200，canonical 是否指向自身。
2. 检查 `robots.txt`、`wp-sitemap.xml`，确认目标 URL 仍在 sitemap 且没有 localhost、测试页或错误 URL。
3. 使用品牌词和 P0 核心词搜索，记录是否出现、展示标题、favicon、摘要、可见排名区间和对应落地页。
4. 每天轮换至少 8 个 P1 词；每周至少一次覆盖全部 P1/P2 词。
5. 对比 Search Console：点击、展示、CTR、平均排名、查询对应页面、已发现/已抓取未编入索引。
6. 记录关键词是否出现页面竞争：同一查询若多个页面反复切换，检查搜索意图和内链，不立即新建重复页。
7. 只在出现明确异常时提出修改；正常波动写入观察项。

## 评估阈值

- 单日变化：只记录。
- 连续 3 天目标页不可访问、canonical 错误或 sitemap 消失：立即修复。
- 连续 7 天展示增长但 CTR 低：检查 title 和 description。
- 连续 14 天有展示但排名 20–50：优先补内容深度、相关内链和真实资料。
- 连续 14 天无展示：检查收录、搜索意图、页面质量和词是否值得继续。
- 任何错误认证、虚假能力或不真实地域信息：立即删除或改正，不以排名为理由保留。
