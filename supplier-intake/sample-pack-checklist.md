用途：向工厂索取样品包、建立产品资料库、拍官网/社媒素材、客户初步沟通。

## 样品标签格式

每个样品贴标签，格式如下：

```text
Sample ID: SBS-001
Material: EPDM closed-cell foam
Thickness: 3mm
Hardness / Density: 30 kg/m³
Adhesive: None
Process: Die cutting
Application: Outdoor electrical enclosure gasket
Supplier: [工厂名称]
Date Received: YYYY-MM-DD
```

## 样品编号规则

```text
SBS-[三位序号]

示例：
SBS-001  EPDM 3mm 泡棉垫片
SBS-002  硅胶 2mm 垫片
SBS-003  背胶 EPDM 模切垫
SBS-004  D 型柜门密封条
...
```

## 收到样品后的动作

- [ ] 贴 Sample ID 标签并拍照存档
- [ ] 录入下方编号表
- [ ] 拍产品图（白底 / 卡尺测量 / 按压回弹）
- [ ] 确认材料 TDS 是否可索取
- [ ] 标记是否可用于官网公开展示（无客户 logo / 无保密图纸）
