# 工厂资料与素材索取清单

用途：筛选合作工厂、建立产品资料库、制作官网与 Facebook 内容、接到询盘后快速报价。

原则：先索取现成资料，不要求工厂专门制作；所有公开素材必须无客户 Logo、无保密图纸、无客户订单信息，并取得工厂同意。

## 第一批：必须提供

### 1. 工厂与联系人信息

- [ ] 公司全称、工厂地址、成立年份
- [ ] 主要联系人、手机/微信、邮箱
- [ ] 主营产品与最擅长的 3–5 类产品
- [ ] 主要出口市场
- [ ] 可配合的贸易方式：EXW / FOB / CIF

### 2. 产品能力

- [ ] 可做材料：EPDM、硅胶、NBR、CR、EVA、PU 等
- [ ] 可做工艺：模切、模压、挤出、背胶、接角/接框等
- [ ] 常规厚度、硬度、密度、颜色范围
- [ ] 可用背胶品牌与型号，包括 3M 和普通背胶
- [ ] 最小打样数量、量产 MOQ
- [ ] 样品交期、模具/刀模周期、量产交期
- [ ] 常规包装方式

### 3. 产品实拍素材

- [ ] 10–20 张清晰的产品实拍图，尽量包含不同材料和工艺
- [ ] 3–5 段竖屏短视频，每段 10–30 秒
- [ ] 尺寸测量、厚度测量、硬度或回弹展示
- [ ] 背胶撕膜、粘贴或安装演示
- [ ] 密封条截面、接角、接框和细节特写
- [ ] 包装与装箱照片

素材要求：原图或原视频，不加微信水印；背景尽量整洁；不出现客户 Logo、客户图纸、订单标签和其他保密信息。

### 4. 工厂与工艺素材

- [ ] 工厂门头或外景 1–2 张
- [ ] 整洁的车间全景 2–3 张
- [ ] 模切、模压、挤出、背胶等主要设备照片
- [ ] 生产过程短视频 3–5 段
- [ ] 检验过程：卡尺、硬度计、拉力或其他测试
- [ ] 仓库、原材料和成品区照片

### 5. 基础文件

- [ ] 营业执照（仅用于内部核验）
- [ ] 公司或产品目录 PDF
- [ ] 主要材料 TDS
- [ ] 可提供的 SDS
- [ ] 有效期内的 RoHS / REACH 报告
- [ ] ISO 等工厂体系证书（如有）
- [ ] UL94 或其他材料等级资料（如有）

## 第二批：方向匹配后再提供

### 6. 样品包

- [ ] EPDM 闭孔泡棉垫片：2mm、3mm、5mm 各 1 件
- [ ] 硅胶实心或泡棉垫片：2 件
- [ ] 背胶模切垫片：普通背胶和 3M 背胶各 1 件
- [ ] D 型、P 型、U 型或常规柜门密封条：各 30cm 左右
- [ ] EPDM、硅胶、NBR、EVA、PU、CR 材料样块
- [ ] 不同硬度或密度的对比样块 2–3 件
- [ ] 异形定制件 1–2 件，用于展示工艺能力
- [ ] 常规出口包装样品 1 件

每个样品需标注：材料、尺寸/厚度、硬度或密度、背胶型号、生产工艺和典型应用。

### 7. 典型案例信息

- [ ] 3–5 个可公开的匿名案例
- [ ] 每个案例说明：应用行业、客户问题、材料选择、结构/工艺方案和结果
- [ ] 配套产品图或安装图
- [ ] 如不能披露客户名称，统一使用匿名描述，例如“欧洲电气柜制造商”

严禁提供客户保密图纸、客户名称、采购数量和可识别订单信息。

## 公开使用授权确认

请工厂用文字确认：

```text
以上指定的产品、车间、设备及工艺图片/视频不含客户保密信息，同意 SealBridge Supply 用于官网、Facebook、LinkedIn、产品介绍及海外客户沟通。未经另行确认，不公开工厂名称、联系方式、证书编号或其他敏感信息。
```

## 可直接发给工厂的微信话术

```text
您好，我这边正在整理 SealBridge Supply 的产品资料库，用于海外客户开发、官网和 Facebook 内容展示。想先向贵司收集一些现成资料，不需要专门拍摄或制作，有合适的原图/原视频直接发我即可。

第一批希望了解：
1. 贵司最擅长的产品、材料和工艺；
2. 常规规格、MOQ、打样及量产交期；
3. 10–20 张产品实拍图和几段生产/测试短视频；
4. 整洁的车间、设备、检验和包装照片；
5. 产品目录，以及现有的 TDS、SDS、RoHS、REACH、ISO 等资料。

素材请尽量发原图/原视频，不要带微信水印。不要发送客户 Logo、客户图纸、订单标签或其他保密信息。我们公开使用前也会筛选，不公开贵司联系方式和敏感资料。

如果产品方向匹配，下一步我想配一套 EPDM 泡棉、硅胶、背胶模切垫片、柜门密封条和材料样块，用于产品学习及客户初步沟通。样品费用和快递可以按贵司方便的方式安排。

另外，请确认您发给我的指定产品、车间和工艺素材，在不泄露客户及贵司敏感信息的前提下，可以用于 SealBridge Supply 官网、Facebook、LinkedIn、产品介绍及海外客户沟通。谢谢。
```

## 收到资料后的内部检查

- [ ] 文件按“工厂—材料—产品—日期”命名并归档
- [ ] 核对证书主体、材料牌号、报告日期和有效期
- [ ] 标记哪些素材允许公开、哪些仅供内部使用
- [ ] 删除或遮挡客户 Logo、订单号、图纸编号和个人信息
- [ ] 记录工厂回复速度、资料完整度、专业度和配合度
- [ ] 从每家工厂挑选 3–5 条可用于 Facebook 的真实内容线索
