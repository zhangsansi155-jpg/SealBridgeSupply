# 供应商微信沟通话术

## 第一轮：确认方向 + 要实拍图

```text
您好，我这边主要做海外 B2B 客户开发，目前准备主打电气箱、控制柜、户外设备用定制密封件。

产品方向先定在：电气箱密封垫、控制柜密封条、EPDM 泡棉垫、硅胶垫片、背胶模切垫片、按图定制密封件。

我现在主要不是问单个产品价格，而是想先了解贵司这几个方向是否匹配。方便的话，您可以先发一些类似产品的实拍图/视频吗？产品图、样品图、车间图、安装演示图都可以，不带客户 logo 或保密图纸就行。

如果方向合适，我想后续配一套样品包，建立产品资料库。以后有海外客户询盘，我会把图纸、材料、数量、证书要求整理清楚后再发您报价，尽量减少反复沟通。
```

## 第二轮：确认工艺与材料

```text
感谢分享。想再确认几个方向：

1. 贵司主要做模切、模压、还是挤出？
2. EPDM 泡棉、硅胶、NBR 这几类材料是否都有稳定供应？
3. 背胶模切（3M 或普通背胶）是否可做？
4. 小批量打样一般 MOQ 和交期大概多少？
5. RoHS、REACH、TDS、SDS 这些资料是否方便提供？
```

## 第三轮：索取样品包

```text
方向看起来比较匹配。我想先配一套样品包，用于产品学习和海外客户初步沟通。样品包希望包含：

- EPDM 泡棉垫片（2/3/5mm 各一）
- 硅胶垫片
- 背胶模切垫片
- 柜门密封条（D/P/U 型样段）
- 几种材料样块（EPDM、硅胶、NBR、EVA 等）

每个样品麻烦贴一下材料、厚度、硬度/密度、工艺说明。样品费用和快递方式您看怎么安排方便？
```

## 注意事项

- 不要一上来就问最低价
- 不要发送客户保密图纸
- 车间图优先选整洁、设备清晰的
- 记录每家工厂的响应速度、英文沟通能力、文件配合度
