# 工厂沟通与配合内容清单

适用场景：筛选工厂、建立供应商池、准备官网产品资料、向工厂询价、做样品包、后续承接海外客户询盘。

核心原则：先判断工厂是否适合我们的 6 类产品方向，再谈具体报价。不要一上来只问最低价。

---

## 1. 我们要先向工厂说明的定位

可以直接发给工厂：

```text
您好，我们这边主要做海外 B2B 客户开发，官网方向是电气箱、控制柜、户外设备、电子外壳相关的定制密封件。

目前主打 6 类产品：

1. 电气箱密封垫
2. 控制柜密封条
3. EPDM 泡棉垫
4. 硅胶垫片
5. 背胶模切垫片
6. 按图定制橡胶密封件

我们现在想先筛选几家匹配的工厂，建立产品资料、样品包、报价流程和后续项目配合机制。以后有海外客户询盘，我们会把图纸、材料、数量、应用场景和证书要求整理清楚后再发您确认。
```

---

## 2. 第一轮要确认：工厂是否匹配

向工厂确认这些问题：

```text
1. 贵司主要做哪类产品？
2. 是否做电气箱、控制柜、机柜、配电柜、接线盒、户外外壳相关密封件？
3. 主要工艺是模切、模压、挤出、发泡、背胶复合，还是几种都做？
4. 是否支持来图定制、来样定制？
5. 是否接小批量样品？
6. 是否有出口经验？
7. 是否能提供英文资料或配合我们整理英文参数？
8. 是否能提供 RoHS、REACH、UL94、TDS、SDS、ISO 等资料？
9. 是否能提供不带客户 logo 的产品实拍图、视频、车间图？
10. 是否接受我们作为海外客户对接方长期合作？
```

判断标准：

| 项目 | 合格表现 | 不合格表现 |
|---|---|---|
| 产品匹配 | 有密封垫、密封条、泡棉、硅胶、背胶模切经验 | 只做普通杂件、玩具件、日用品 |
| 工艺匹配 | 模切、挤出、模压至少占一种优势 | 什么都说能做但没有案例 |
| 响应速度 | 半天内能回复基本问题 | 两三天不回或只回“可以做” |
| 资料配合 | 能给图、视频、参数、材料说明 | 只给报价，不愿提供资料 |
| 合规意识 | 能说明资料来源和适用范围 | 什么证书都说有，但拿不出来 |

## 4. 需要工厂提供的图片和视频

图片非常重要，后续用于：

```text
内部产品库
客户初步沟通
样品包说明
官网素材参考
业务员培训
```

注意：**不要使用带客户 logo、客户图纸、客户包装、客户项目编号的图片。**

### 4.1 产品图片

每类产品至少要：

```text
主图 1 张
侧面角度图 1-2 张
细节图 1-2 张
厚度/截面图 1 张
背胶/离型纸图 1 张
安装场景图 1 张

```

图片要求：

```text
背景干净
光线清楚
不要水印
不要客户 logo
不要模糊
不要只发电商压缩小图
最好原图发送
```

### 4.2 视频

建议让工厂提供：

```text
产品旋转展示视频
手撕背胶演示视频
压缩回弹演示视频
密封条截面展示视频
模切生产短视频
挤出生产短视频
包装发货视频
车间设备视频
```

视频要求：

```text

不要拍客户订单标签
不要露出客户图纸

```

## 5. 样品包配合清单

建议每家重点工厂先配一套样品包。

### 5.1 基础样品包

```text
EPDM 泡棉垫片：2mm / 3mm / 5mm
硅胶垫片：常用硬度 40A / 50A / 60A
背胶模切垫片：普通胶 / 3M 胶各 1-2 款
控制柜密封条：D 型 / P 型 / U 型 / 包边条样段
电气箱密封垫：闭孔泡棉或 EPDM 样件
定制橡胶件：不涉密的样件 2-3 个
材料样块：EPDM / Silicone / NBR / CR / EVA / PU foam
```

### 5.2 每个样品必须标注

```text
材料
厚度或截面
硬度或密度
工艺
是否背胶
背胶型号
用途建议
是否可提供证书
```

### 5.3 样品包要问工厂

```text
样品费多少？
快递费多少？
能否开样品清单？
能否每个样品贴标签？
能否提供对应电子版参数？
后续客户要样品时最快几天发？
```

## 6. 证书和文件配合清单

不要让工厂笼统说“都有证书”。要让工厂明确：

```text
证书是工厂证书，还是材料供应商证书？
证书是否在有效期内？
证书对应哪种材料？
证书是否对应当前产品？
能否给英文版？
能否给客户查看？
是否可以用于出口项目资料？
```

### 6.1 常见文件

| 文件 | 用途 | 工厂要说明 |
|---|---|---|
| ISO 9001 | 质量体系参考 | 是否工厂主体证书 |
| RoHS | 限制有害物质 | 对应材料还是产品 |
| REACH | 欧盟化学品要求 | 是否为材料报告 |
| UL94 | 阻燃等级 | V-0 / V-1 / HB，具体材料 |
| TDS | 技术数据表 | 材料性能参数 |
| SDS | 安全数据表 | 化学安全信息 |
| FDA/LFGB | 食品接触 | 我们当前不是重点 |
| 材质证明 | 材料确认 | 是否可随货提供 |

### 6.2 证书回复模板

可以发给工厂：

```text
麻烦您把目前可以提供的资料列一下：

1. ISO 证书
2. RoHS
3. REACH
4. UL94
5. TDS
6. SDS
7. 材质证明

每个资料麻烦注明：对应材料、有效期、中文/英文版本、是否可以给海外客户查看。
```

## 8. 工艺能力确认

```text
能切哪些材料？
最大尺寸是多少？
最小孔径是多少？
公差能做到多少？
是否能半断/全断？
是否能贴 3M/Nitto/Tesa 胶？
是否能做卷料/片料？
是否能做小批量？
```

适合产品：

```text
EPDM 泡棉垫
背胶模切垫片
硅胶平垫
电气箱密封垫
```

### 8.2 挤出密封条厂

重点问：

```text
可做 EPDM / PVC / Silicone 吗？
可做发泡还是实心？
是否有现成截面？
是否能按图开模？
开模费多少？
MOQ 多少米？
是否能贴背胶？
是否能裁切定长？
```

适合产品：

```text
控制柜密封条
机柜门边条
配电柜密封条
U 型包边条
D/P 型密封条
```

### 8.3 模压橡胶厂

重点问：

```text
可做哪些材料？
模具费多少？
最小订单量多少？
样品周期多久？
是否能控制毛边？
是否支持嵌件？
是否能做硅胶/EPDM/NBR？
```

适合产品：

```text
硅胶垫片
定制橡胶垫
复杂形状橡胶件
按图定制密封件
```

## 9. 保密和素材使用要求

必须和工厂说清楚：

```text
不要发客户保密图纸
不要发带客户 logo 的产品
不要发带客户标签的包装照片
不要发有客户项目编号的图片
不要发不能公开展示的视频
```

如果图片用于官网或业务资料，要确认：

```text
这个图片是否可以公开展示？
是否有客户 logo？
是否涉及客户图纸？
是否需要打码？
是否可以用于海外客户沟通？
```

建议话术：

```text
我们只需要可公开展示的通用产品图、样品图、车间图，不要客户保密项目、客户 logo、客户图纸和客户包装标签，避免后续合规风险。
```

## 10. 工厂评分表

每家工厂按 100 分评分。

| 维度 | 分值 | 说明 |
|---|---:|---|
| 产品匹配度 | 20 | 是否覆盖我们 6 类产品 |
| 工艺能力 | 15 | 模切/挤出/模压/背胶是否清楚 |
| 图片视频配合 | 10 | 是否愿意提供干净素材 |
| 样品配合 | 10 | 是否愿意做样品包和小批量 |
| 文件证书 | 10 | RoHS/REACH/UL94/TDS/SDS 等 |
| 报价速度 | 10 | 回复是否及时、报价是否完整 |
| 质量意识 | 10 | 是否懂公差、材料、包装、检验 |
| 出口经验 | 5 | 是否有外贸或出口配合经验 |
| 沟通稳定性 | 5 | 是否长期有人对接 |
| 保密意识 | 5 | 是否懂客户图纸和 logo 保护 |

评级：

```text
85-100：A 类重点合作
70-84：B 类可合作
60-69：C 类备用
60 以下：暂不合作
```

## 13. 最终筛选建议

供应商池不要只找一种厂，要分三类：

```text
A 类：密封条/挤出厂
负责控制柜密封条、机柜条、U 型条、D/P 型条。

B 类：模切/泡棉/背胶厂
负责 EPDM 泡棉垫、电气箱密封垫、背胶模切垫片。

C 类：硅胶/模压橡胶厂
负责硅胶垫片、按图定制橡胶件、复杂形状橡胶件。
```

每类至少保留：

```text
2 家 A 类重点工厂
2 家 B 类重点工厂
2 家 C 类重点工厂
```

这样后续客户询盘不会被单一工厂能力限制。
